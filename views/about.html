Esta pagina es una galeria propuesta como trabajo inicial para la clase de plataformas empresariales

Esto implica que es una pagina estatica que no hace mucho ademas de ser, bueno eso estatica

Aun asi, esta pagina cumple como un SPA (Single Page Application) Desde que decide que usaria vanilla javascript para simular apps hechas en svelte o react

Por desgracia, ser reactivo como en svelte o manejar virtual dom es dificil, pero gracias a internet encontre muchas formas de hacer una SPA que funcione
